# EZBook

EZBook is a full-stack booking platform that allows customers to schedule appointments with independent service providers. The platform helps providers manage services, availability, and appointments through an easy-to-use dashboard.

## Live Demo
🔗 [EZBook Website](https://ez-book-green.vercel.app/)

---

# Features

### Customers
- Create an account and log in
- Browse provider services
- Book appointments
- Reschedule or cancel appointments
- View booking history

### Providers
- Manage services
- Set availability
- Accept or reject appointments
- Block off unavailable time
- View upcoming and past bookings

### Scheduling
- Dynamic appointment slots
- Prevents overlapping bookings
- Supports custom service durations
- Automatic buffer times between appointments
